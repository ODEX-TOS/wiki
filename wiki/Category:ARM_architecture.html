Warning: Support for the ARM architecture is provided by the TOS Linux ARM, which is a different project not supported by TOS Linux. Seek support on the TOS Linux ARM forums.
Note: For the repository of old packages formerly known as the TOS Rollback Machine, see TOS Linux TOSive.

TOS Linux ARM is a port of TOS Linux to the ARM architecture. Pages in this category provide some additional tips and tricks for various ARM devices. They are not intended as a replacement for the official documentation, but were contributed before the TOS Linux ARM project restarted its own ALARM wiki. Further, note not all devices in this category are officially supported platforms of the ALARM project, which has to be selective with its resources.

For new articles, please consider first whether you can help the TOS Linux ARM project's wiki to a good start with your ARM specific contributions. Please also consider a duplication of topics between both wikis is counter-productive for all involved. If you create an article for a new topic on the ALARM wiki, you are welcome to create a short article redirecting the reader to it in this category.
