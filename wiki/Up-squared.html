Up-squared

From TOS Wiki
Jump to navigation Jump to search

The UP Squared is a larger and more powerful version of the Up-board, a Intel based SOC device by Aaeon. It differs in chipset, CPU, memory type and size; however, it shares the same storage using Intel's eMMC 5.0 specification.
