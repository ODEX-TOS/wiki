The GNU Compiler Collection (GCC) is part of the GNU toolchain and includes front ends for C and C++.

Installation

Install the gcc package.

Other available front-ends are:

  • gcc-ada for Ada
  • gcc-d for D
  • gcc-fortran for Fortran
  • gcc-go for Go
  • gcc-objc for Objective-C
