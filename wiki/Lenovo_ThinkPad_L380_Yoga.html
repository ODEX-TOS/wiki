Additional Functionality

Install iio-sensor-proxy for automatic screen rotation and brightness adjustment under GNOME.

The thinkpad-yoga-scripts-gitAUR package doesn't work well, because the TouchPad and TrackPoint manufacturer was changed. Using https://github.com/ffejery/thinkpad-l380-yoga-scripts instead works. See makepkg on how to build it. There's also an AUR: thinkpad-l380-yoga-scripts-gitAUR. Remember to edit the /opt/thinkpad-l380-yoga-scripts/rotate/thinkpad-rotate.py as hinted in the readme, because you will probably want the Touchscreen and Pen to rotate together with the screen.
