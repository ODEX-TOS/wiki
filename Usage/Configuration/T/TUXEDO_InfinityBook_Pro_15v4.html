Drivers

See Intel Graphics and Hardware Acceleration

Fn Function Keys

The most Fn function keys work out of the box. For the Fn function keys F10, F11 and F12 the following must be set as kernel parameter 

acpi_os_name=Linux acpi_osi= acpi_backlight=vendor

Keyboard Backlight

TUXEDO Computers provides a driver for their models with RGB keyboard. TUXEDO Keyboard can be installed manually from source or you use tuxedo-keyboardAUR

Eye candy

Plymouth Theme

TUXEDO Computers provides their own Plymouth Theme named "TUXEDO Plymouth One". The "TUXEDO Plymouth One" Theme can be installed manually from source or you use tuxedo-plymouth-oneAUR
