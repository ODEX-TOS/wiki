MSI PS63 Modern Optimus Yes Yes Yes Yes Yes n/a BisonCam integrated webcam works out of the box with Cheese just needs to be activated with Fn + F6. All works out of the box with newer kernel above 5.x. However touchpad is disable when wake up from suspend. Needs to reset i2c bus. Add a custom script in

/lib/systemd/system-sleep/restart_trackpad 

#!/bin/sh
case "$1" in
    post)
            echo -n "i2c_designware.0" > /sys/bus/platform/drivers/i2c_designware/unbind
            echo -n "i2c_designware.0" > /sys/bus/platform/drivers/i2c_designware/bind
    ;;
esac
