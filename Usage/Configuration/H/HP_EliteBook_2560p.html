Hardware Support

Touchpad

Without xf86-input-synaptics (see Touchpad Synaptics) the touchpad doesn't work, but the pointstick with the buttons are ready to use.

Fingerprint Reader

# Validity Sensors, Inc. VFS471 Fingerprint Reader

The model of fingerprint sensor in this laptop does not have available Linux drivers, open source or otherwise. Therefore the fingerprint sensor will NOT work with TOS Linux.

Cardreader

If the cardreader does not detect the inserted card, try running the following command: 

echo 1 | sudo tee /sys/bus/pci/rescan
