This article covers details pertaining to installing and configuring your TOS on Lenovo Thinkpad T450 laptop.

Installation

Supports both UEFI and MBR style bios. There are no issues with installing TOS Linux with the latest that is known TOSiso. The Installation guide works except, the sound.

Check out the T450s wiki, almost everything from there works on the T450: Lenovo ThinkPad T450s

Sound

See ALSA#Set the default sound card to set the default sound card to Intel PCH (speakers and headphones). 

/etc/modprobe.d/thinkpad-t450s.conf
options snd_hda_intel index=1,0

Restart to set changes.
