Programming languages

From TOS Wiki
Jump to navigation Jump to search

This page attempts to list all programming languages with a compiler / interpreter packaged for TOS Linux.

  • Assembly – as of binutils, fasm, nasm, yasm
  • Ada – GCC
  • AWK
  • BASIC – freebasic
    • AMOS2
    • Gambas
  • C – GCC, Clang
    • C++ – GCC, Clang
    • C# – Mono
    • Objective-C – GCC, Clang
    • Vala – vala
    • Zig – zig
  • Crystal
  • D
  • Dart – dart
  • Forth – 4thAUR, gforthAUR
  • Fortran – GCC
  • Go – go, GCC
  • Java
    • Groovy – groovy
  • JavaScript – rhino, spidermonkeyAUR, v8AUR
    • Node.js
  • Julia
  • Kotlin – kotlin
  • Lua – lua
  • Nim – nim
  • Octave
  • Pascal – fpc
  • Perl
  • PHP
  • Python
  • R
  • Ruby
  • Rust
  • Scala
  • Swift – swiftAUR, swift-binAUR
  • Tcl – tcl

Functional languages

Note that some languages listed above also support some functional programming.

  • Erlang – erlang
    • Elixir
  • Haskell
  • Lisp dialects
    • Clojure
    • Common Lisp
    • Emacs Lisp
    • Scheme
  • ML
    • Standard ML – smlnj, mlton, mosmlAUR, smlsharpAUR
    • OCaml – ocaml

Shell languages

Bash is a dependency of the base meta package. For others, see Command-line shell.
